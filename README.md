### `Full stack web application, Express.js & React.js`


### `1. Clone the repository with git clone`
### `2. Step into the local directory 'react-express-app`
### `3. Run 'npm i`
### `4. Run 'nodemon server.js`
### `5. Step into 'client'`
### `6. Run 'npm i'`
### `7. Run 'npm start'`

### `http://localhost:3000/`
